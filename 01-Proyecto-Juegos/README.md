El presente documento se trata de la documentación que recoge la información del proyecto sobre juegos realizados con el lenguaje de programación de Python. Se tratará del desarrollo de dos juegos muy conocidos: piedra, papel o tijeras y juego de preguntas geográficas. Vamos a describir en qué consisten cada uno de ellos, para tener una idea clara de qué lógica computacional es la mejor para cada uno de nuestros códigos.
